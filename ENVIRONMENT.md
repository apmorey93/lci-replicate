# Environment

- OS: Linux/macOS/Windows (tested on Linux x86_64)
- TeX: TeX Live 2023+ (latexmk, pdflatex)
- Python: 3.10+
- Python packages: see requirements.txt

## Quick start
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
bash scripts/rebuild_all.sh
```
